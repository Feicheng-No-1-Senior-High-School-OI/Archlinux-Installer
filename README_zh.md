## Feature

- 伪界面，比纯命令行更加友好
- 支持KDE、Xfce、Gnome桌面系统的安装
- 支持Gnome桌面系统的配置和中文输入法(搜狗)的安装
- 支持Gnome下常用软件配置的备份和恢复(可以添加自己的备份/恢复脚本)
- 支持BIOS和UEFI两种启动方式
- 自动检测Windows启动项

## How To Use

首先，需要准备好Arch Linux启动盘，并连网。

使用安装器：

```bash
bash <(curl https://raw.githubusercontent.com/si9ma/Archlinux-Installer/master/install.sh)
```
